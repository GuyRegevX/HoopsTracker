How to Update Game Statistics

  • Select a team from the dropdown menu
  • Choose a player from that team's roster
  • Pick the type of stat to update (points, assists, etc.)
  • Select the game where this event occurred
  • Changes will appear in the statistics tables below within seconds
WebSocket: ws://localhost:8082/ws/game_live_update
